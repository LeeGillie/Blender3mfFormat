The original add-on supported Blender 2.8 through 4.2 and provided comprehensive support for the 3MF Core Specification v1.2.3.

**Note:** The 3MF specification has been updated to v1.4.0 (February 2025). This fork currently maintains v1.2.3 support. For information on newer specification features and planned updates, see [3MF-SPEC-UPDATES.md](3MF-SPEC-UPDATES.md).

**Maintainer:** Lee Gillie, CCP  
**GitHub:** https://github.com/LeeGillie

This fork was created to update the extension for Blender 5.x compatibility:

### Major Changes
- Updated to Blender 5 extension format using `blender_manifest.toml`
- Restructured project to use new extension packaging system
- Added VS Code development environment setup
- Created build scripts for easier distribution
- Updated documentation for Blender 5 workflow

### Version History
- **v2.0.0** (2026) - Blender 5 compatibility fork
  - New extension manifest format
  - Updated for Blender 5 API
  - Added development tooling
  
- **v1.0.2** (Original by Ghostkeeper)
  - Full 3MF Core Specification 1.2.3 support
  - Support for Blender 2.8-4.2
