# GitHub Release Commands - v2.2.1

## Step 1: Commit Changes

```powershell
git commit -m "Release v2.2.1: GPL-3.0 upgrade and Triangle Sets compliance fixes

- Upgraded license from GPL-2.0 to GPL-3.0-or-later (Blender requirement)
- Fixed Triangle Sets namespace (now uses extension namespace)
- Added required identifier attribute to triangle sets
- Fixed non-geometric nature (removed material generation)
- Added namespace declaration to model element
- Removed random color generation
- Updated all documentation
- Removed internal files from repository
"
```

## Step 2: Push to GitHub

```powershell
git push origin main
```

